Usage - Extensions

For example lets instantiate a barchart-library:

You installed the library named 'barchart' as extension.

So - you can build an object out of it with:

$chart = $base->getExtension('barchart');

In order to play with it you have only this to do - nothing else - the object the exists as $chart!

With this object you can do everything what this object can do.

Maybe $chart->buildBarChart();

Thats all!
