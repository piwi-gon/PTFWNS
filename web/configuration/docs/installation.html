Installation

There is no installer-script at this time - you have to copy the basic-system to a path where you want to install it.

The main reason why there is no installer is: i got not enough time for now.

The next release will have an installer and you are able to update the framework with the builtin frontend.

There will be a checker for the dependencies too, hopefully.

You are also be able to install modules and extensions.
These installations are for now not really checking any dependencies but in one of the next releases there will be a check if necessary.

Thats all for now.
Enjoy the framework and be free to use and extend it ...
