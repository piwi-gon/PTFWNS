Development - Extensions

If you want to install a new Extension or update an already installed one you should use this frontend for this.
It's very simple - just click on New-Button () for installing a new one
or the Update-Button () for updating an extension.

And then - follow theinstructions.

After installing/updating the extension you can use it like this:

$obj = $base->getExtension('newExtension');

Sometimes it is necessary to create a wrapper around the external-library to work with the framework.
But this is very simple:
Create a simple class which is calling the library and set this class to be the library to be included by the framework - thats all!
