Usage - Modules

For example lets instantiate the date-module:

You installed the module named 'Date' as module.

So - you can build an object out of it with: $oDate = $base->getDate();

In order to play with it you have only this to do - nothing else - the object the exists as $oDate!

With this object you can do everything what this object can do. Maybe $oDate->getDate();

Thats all!
