Usage - General

In general - to use this framework notheng else is to do as to install it.
Next thing is to instatiate the base-class.

From this point you can do everything you want - with every module and/or extension you want.
$obj = $base->get{ModuleName}();
or
$obj = $base->getExtension('{ExtensionName}');

That's it - nothing else for usage. See basics for more details.
